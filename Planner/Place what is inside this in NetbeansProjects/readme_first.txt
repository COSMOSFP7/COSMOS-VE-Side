
Folder of the NetBeans 8.0.2 Maven JAVA SE Project. Make sure your NetBeans installation has Maven and open the project. It should compile with no problems once maven locates and downloads alla dependencies. Use the option "build with dependencies" after right clicking on the name of the Project in the Project explorer.
The folder target contains the .jar file and the lib folder with all dependencies needed.

In order to run the code, start cmd (command line), go to the path of the .jar file (command is: cd path_to_jar), execute "java -jar nameofjar.jar" and it will be running showing also a list of logs for your convenience while performing actions.
